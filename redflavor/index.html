Contributions

Rubbr
Build tool for large LaTeX projects.
Implemented in Ruby.
Awesomer
A library for interacting with the awesome window manager.
Implemented in Ruby.
Reprise
Minimalistic blog engine with hAtom support.
Implemented in Ruby.
Generates static files, so scaling and performance is a non-issue.
Sinatra based version also available.
Accomplish
Minimalistic task list.
Implemented in Ruby.
Mofo
Microformat parser.
Contributions for correct parsing of hCard authors in hAtom entries.
Implemented in Ruby.
Acts as Authentable
Ruby on Rails authentication plugin.
Port of restful_authentication to include BCrypt secure password storage.
Seoul
Vim color scheme for the 88 color mode of rxvt-unicode .
Its look can be seen in a screenshot taken while editing Ruby code.
