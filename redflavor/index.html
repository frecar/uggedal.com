Rubbr
Build tool for large LaTeX projects.
Implemented in Ruby.
Awesomer
A library for interacting with the awesome window manager.
Implemented in Ruby.
Reprise
Minimalistic blog engine with hAtom support.
Implemented in Ruby.
Generates static files, so scaling and performance is a non-issue.
Sinatra based version also available.
merb_cache_observer
Merb plugin which observes DataMapper models for changes and expires caches when they are updated.
Implemented in Ruby.
Django notices
Django messages without the db or cookies.

Implemented in Ruby.
